# OCSplus Docs

本目录是 OCSplus 项目的 Agent 协作大脑。

## 核心文档

- `PRD.md`：产品需求
- `Tasks.md`：任务清单
- `Progress.md`：进度记录

## 专项文档

- `Research.md`：开源项目研究
- `Architecture.md`：架构设计
- `UI.md`：界面设计
- `Review.md`：审查记录

## Agent 协作文档

- `Agents.md`：Agent 分工
- `CodexPrompt.md`：Developer/Codex 固定提示词
- `DailyWorkflow.md`：每日工作流

## 使用原则

1. PM Agent 只维护任务和进度，不写代码。
2. Developer Agent 一次只做一个任务。
3. Reviewer Agent 审查后再进入下一步。
4. 任何新增需求必须先进入 PRD 和 Tasks。
